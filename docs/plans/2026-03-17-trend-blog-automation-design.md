# 트렌드 기반 블로그 자동화 시스템 설계

## 날짜: 2026-03-17

## 개요

"정보 전달"이 아니라 **"이걸 왜 알아야 하는지 알려주는 콘텐츠"**를 자동으로 생성하고
네이버 블로그에 발행하는 시스템.

뉴스, 경제, 핫이슈(연예/사회/트렌드)를 수집하여 일반인이 이해하기 쉬운 형태로
재구성하고, 내 삶/돈/감정에 미치는 영향을 해석해주는 블로그 글을 자동 생성한다.

## 파이프라인 (8단계)

### STEP 1. 트렌드 수집
- 네이버 뉴스 검색 API (정치/사회/IT/생활)
- 네이버 검색 API (경제 키워드 실시간)
- 네이버 실시간 급상승 검색어 + 연예뉴스
- 중복 제거 및 트렌드 점수 산정

### STEP 2. 궁금증 포인트 추출 (AI)
- "이게 왜 화제야?"
- "나한테 어떤 영향이 있어?"
- "앞으로 어떻게 될까?"
- "사람들이 가장 궁금해할 3가지"

### STEP 3. 클릭 유도 제목 생성 (AI)
- 3개 후보 생성 → 사용자 선택
- 검색 유입 + 호기심 유발 최적화

### STEP 4~6. 콘텐츠 생성 (AI 핵심)
글 구조:
1. 도입부 — 독자 공감 유도
2. 핵심 요약 카드 — 3줄 요약
3. 쉽게 풀어쓴 본문 — 전문용어 → 일상 언어
4. 중간 요약 카드 — "여기까지 정리하면..."
5. 내 삶에 미치는 영향 — 돈/생활/감정
6. 앞으로 어떻게 될까? — 전문가 의견 + AI 분석
7. 최종 핵심 요약 — "한 줄로 말하면..."

### STEP 7. 이미지 자동 생성
- 썸네일: 제목 + 핵심 키워드 기반
- 요약 카드: 핵심 3줄 요약 이미지
- 상황 이미지: 본문 중간 삽입용

### STEP 8. 블로그 최적화 + 발행
- SEO 키워드 배치, 소제목 구조화
- HTML 포맷팅 (네이버 블로그 호환)
- 미리보기 → 사용자 승인 → 네이버 블로그 API 발행

## 기술 스택

| 구성요소 | 기술 | 비용 |
|---------|------|------|
| 언어 | Python 3.11+ | 무료 |
| 트렌드 수집 | 네이버 검색 API | 무료 (하루 25,000건) |
| 키워드 분석 | 네이버 검색광고 API | 무료 |
| 글 작성 AI | OpenAI GPT-4o-mini | ~월 $2~3 |
| 이미지 생성 | DALL-E 3 / Pollinations.ai | DALL-E: 건당 $0.04 / 무료 대안 |
| 블로그 발행 | 네이버 블로그 오픈 API | 무료 |

예상 월 운영비: $3~10 (하루 2~3건 기준)

## 사전 준비

| API | 발급 위치 | 용도 |
|-----|----------|------|
| 네이버 오픈 API | developers.naver.com | 뉴스 검색 + 블로그 발행 |
| 네이버 검색광고 API | searchad.naver.com | 키워드 검색량 분석 |
| OpenAI API Key | platform.openai.com | 글 작성 + 이미지 생성 |

## 프로젝트 구조

```
blog-automation/
├── main.py                  # 메인 실행 파일
├── .env                     # API 키 (비공개)
├── config.py                # 설정값
├── modules/
│   ├── trend_collector.py   # STEP 1: 트렌드 수집
│   ├── curiosity_engine.py  # STEP 2: 궁금증 포인트 추출
│   ├── title_generator.py   # STEP 3: 제목 생성
│   ├── content_writer.py    # STEP 4-6: 콘텐츠 생성
│   ├── image_generator.py   # STEP 7: 이미지 생성
│   └── blog_publisher.py    # STEP 8: 블로그 발행
├── prompts/
│   ├── content_prompt.txt   # 글 생성 프롬프트 템플릿
│   ├── title_prompt.txt     # 제목 생성 프롬프트
│   └── curiosity_prompt.txt # 궁금증 추출 프롬프트
├── templates/
│   └── blog_template.html   # 네이버 블로그 HTML 템플릿
└── requirements.txt
```

## 콘텐츠 철학

- 정보 전달이 아닌 "이걸 왜 알아야 하는지" 중심
- 전문용어를 일상 언어로 변환
- 내 삶/돈/감정에 미치는 영향 해석 필수
- 글 중간중간 핵심 요약 삽입으로 가독성 확보
